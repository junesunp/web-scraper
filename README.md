# jcrew-scraper
Web Scraper for J.Crew website using Nifi and Scrapy

## Setup

Install required libraries
```
pip install -r requirements.txt
```

## Run a Spider
1. cd into directory of spiders
2. Run the following command
```
scrapy crawl <name-of-spider>
```
